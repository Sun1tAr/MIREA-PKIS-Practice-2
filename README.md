# MIREA-PKIS-Practice-2

## 📋 Описание проекта
Консольное приложение на Java для анализа текстовых файлов. Разработано в рамках практического занятия по дисциплине "Программные компоненты информационных систем" (ПКИС).

## 🎯 Функциональность
Приложение предоставляет интерактивный интерфейс командной строки для работы с текстовыми файлами и поддерживает следующие операции:

| № | Операция | Описание |
|---|----------|----------|
| 0 | Завершить работу | Выход из программы |
| 1 | Ввести путь к файлу | Установка пути к анализируемому файлу |
| 2 | Вывести текст в консоль | Отображение содержимого текущего файла |
| 3 | Найти подстроку в файле | Поиск количества вхождений заданной подстроки |
| 4 | Посчитать количество слов | Подсчёт общего количества слов в файле |
| 5 | Провести анализ набора файлов | Многопоточный анализ нескольких файлов |
| 6 | Сгенерировать случайный набор файлов | Создание тестовых файлов со случайным содержимым |

## 🏗️ Архитектура проекта

```
src/
└── main/
    └── java/
        └── my/
            └── learn/
                ├── config/         # Конфигурационные классы и сообщения
                ├── console/        # Вывод в консоль
                ├── executor/       # Исполнители операций
                ├── model/          # Модели данных и сообщений
                ├── service/        # Бизнес-логика работы с файлами
                ├── util/           # Утилиты (логирование, анализатор)
                └── Main.java       # Точка входа
```

## 🔧 Технологии
- Java 17
- Maven
- Lombok (для сокращения шаблонного кода)
- JUnit 5 (для тестирования)

## 🚀 Запуск приложения
```bash
mvn compile exec:java -Dexec.mainClass="my.learn.Main"
```
или через скомпилированный JAR:
```bash
mvn package
java -jar target/MIREA.PKIS.Practice-2-1.0-SNAPSHOT.jar
```

## 🧪 Тестирование
```bash
mvn test
```
Проект содержит модульные тесты для `FileService` с использованием JUnit 5.

## 📁 Генерация тестовых файлов
Приложение может автоматически создавать тестовые файлы со случайным содержимым в указанной директории (по умолчанию `./test_files`).

## 🔍 Особенности реализации
- **Многопоточный анализ** файлов с использованием пула потоков
- **Интерактивный консольный интерфейс** с меню операций
- **Обработка ошибок** при работе с файловой системой
- **Логирование** работы приложения (опционально)
- **Валидация** путей к файлам
